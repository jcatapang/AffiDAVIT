# XViT-LaDA
Explainable Diffusion-Augmented ViT using Grad-CAM

## Dependencies
- transformers
- torch
- torchvision
- pytorch-grad-cam
- datasets
- pandas
- [Stable Diffusion Web UI](https://github.com/AUTOMATIC1111/stable-diffusion-webui)
